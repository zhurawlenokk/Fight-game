# Fight game

## Описание

- Игра имеет одну страницу, /fight
- UI не важен
- Игра пошаговая

# Юзер флоу

1. Вы делайте запрос на получение двух рандомных героев из API https://rickandmortyapi.com/documentation/#rest
2. Экран делится на две части
3. В каждой части есть

- есть фото героя
- блок рандомайзер 4 X 2
- собственные комбо
- Кнопка Атака
- Кнопка Рерол

### 1. Запрос на получение двух рандомных героев

Нужен для того что бы визуально иметь разницу между героями
В адаптиве делаем flex wrap - будет на мобилке в один столбец все

### 2. Экран делится на две части

Игра пошаговая
Сначала ходите вы, а потом другой игрок
Конец шага - нажатие на кнопку атаки или послений Roll

### 3. Блок рандомайзер

Состоит из цветных квадратов 4 X 2 (можно сделать не цвета, а иконки допустим)
На его основе будут собираться комбо который будут генерироваться отдельно
Комбо максимум 3 на игрока
Минимум 1 квадрат на комбо, максимум 4
При нажатии на кнопку Roll, цвета их рандомазера меняются на другой
Рерол можно сделать максиум 3 раза за ход
Ячейки внутри рандомазера можно зафиксировать нажав на них
Если так сделать, то отдельный зафиксированые квадраты на ролятся, это основной инструмент для формирования комбо

### 4. Кнопка атака

При нажатии на кнопку атака у пользователя за основу берутся цвета которые уже выпали в рандомайзере
Далее цвета из рандомайзера матчатся с комбо вашего персонажа
Одно комбо смотрит на матч из 8 вариаций, если есть соответствие(допустим 2 цвета совпали),матч комбо продолжается из 6
оставшихся цветов
Остальные комбо проверяются так же начиная с 8 вариаций
Каждое комбо отнимает хп персонажа (запроектировать что бы мог быть хил

### Навигация

Делаем хедер и футер с ссылками из схемы + селект со сменой языка
Все это мокаем

### Окончание игры

Показываем диалог окно, победил n игрок

## Пример

    1.  Загрузилась страница
    1.1  Сгенерировались пользователи (запрос на 2 пользака + генерация комбо для каждого)
    2.  Вы ходите первые
    3.  У вас автоматически генерируется квадраты в рандомайзере
    4.  Далее вы фиксируйте 1 квадрат
    5.  Нажимайте Roll, 7 квадратов меняют свой цвет на другой рандомный
    6.  Снимайте фиксацию с предыдущего квадрата, и фиксируете другие 3
    7.  Нажимайте Roll
    8.  3 квадрата не меняются, остальные 5 меняются
    9.  Нажимайте Атаковать (допустим вас рандомайзер устроил)
    10. По одному комбо с верху вниз матчится комбо вашего игрока и то что вам выпало
    11. В зависимости от n количества совпадений у вашего противника отнимается хп
    12. После всех матчей комбо и вашего рандомайзера ход переходит к следующему игроку (см пункт 3)
    13. Побеждает тот кто первый довел до нуля хп противника
    14. Показываем всем модалку что выйграл n пользователь, начать с начала ?
